 * 支持列表
 
   | 支持列表        | 说明                                             |       |
   | :----:          | :----:                                           | :---: |
   | cors            | 支持跨域                                         |       |
   | apollo          | 支持apollo配置中心                               |       |
   | 阿里大鱼        | 阿里短信&邮件提醒                                |       |
   | 阿里云oss       | 支持上传oss                                      |       |
   | 阿里云rds       | 支持阿里云数据库                                 |       |
   | 七牛云          | 支持上传七牛云                                   |       |
   | 又拍云          | 支持上传又拍云                                   |       |
   | 阿里云日志服务  | 支持日志打到阿里云日志服务<br>（互联网项目推荐） |       |
   | Jenkins持续集成 | 支持Jenkins                                      |       |
